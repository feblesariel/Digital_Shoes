## Descripción

Tienda virtual de zapatillas online - trabajo para Sprint 1 & 2 - DigitalHouse.<br><br> 
La tematica se selecconó de ntre varias ideas como:<br><br>
- Geofertas: Geolocalizacion de ofertas y promociones a su alrededor.
- Gestión turística: Página enfocada a personal interesado en actividades de ocio, aventura y deportes.
- eComerce de venta de articulos deportivos.
- eComerce de venta de autos.
<br><br>
La seleccionada debe permitir vitrina de productos, registro de usuarios, carrito de compras donde el cliente puede realizar la seleccion de cada uno de los items, filtrar para buscar alguno de su preferencia adicionado de una vista detallada con las diferentes caracteristicas de cada uno de los productos 


## Iniciar aplicación

Install
```
  cd 'su carpeta de proyecto'
  npm install express
  git init
  git clone https://github.com/feblesariel/Digital_Shoes.git
    
```
Start


```
    node app.js
```
La web corre en servidor local.<br />
Open [http://localhost:3030](http://localhost:3030) para verla en el navegador.

## El proyecto 
<br><br>
Se trata sobre la maquetación y creación de un sitio web para una tienda de zapatillas deportivas. En él vamos a ver la evolución que hemos ido llevando para la puesta en marcha de la tienda online.<br>El objetivo principal es que la tienda online este operativa en todas las funciones, desde el proceso de selección del producto, pasando por el proceso de compra, hasta llegar al paso final, el pago.<br> Otro de los objetivos es intentar conseguir a lo largo de los sprints que la tienda online vaya evolucionando y sobretodo incrementando las funciones. La tienda online nos ofrece la posibilidad de acercarnos al mundo real.
