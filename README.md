## Descripción

Tienda virtual de zapatillas online - trabajo para Sprint 2 - DigitalHouse


## Iniciar aplicación

Install
```
  cd 'su carpeta de proyecto'
  npm install express
  git init
  git clone https://github.com/feblesariel/Digital_Shoes.git
    
```
Start


```
    node app.js
```
La web corre en servidor local.<br />
Open [http://localhost:3030](http://localhost:3030) para verla en el navegador.

## El proyecto 

Se trata sobre la maquetación y creación de un sitio web para una tienda de 
zapatillas deportivas.<br>En él vamos a ver la evolución que hemos ido llevando para la puesta en marcha de la tienda online.<br>
El objetivo principal es que la tienda online este operativa en todas las funciones, desde el
proceso de selección del producto, pasando por el proceso de compra, hasta llegar al paso
final, el pago.<br> Otro de los objetivos es intentar conseguir a lo largo de los sprints que la tienda
online vaya evolucionando y sobretodo incrementando las funciones.<br> La tienda online nos ofrece
la posibilidad de acercarnos al mundo real.
