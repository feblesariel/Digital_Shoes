## Descripción

Tienda virtual de zapatillas online - trabajo para Sprint 1 - DigitalHouse


## Iniciar aplicación

Install
```
  cd 'su carpeta de proyecto'
  npm install express
  git init
  git clone https://github.com/feblesariel/Digital_Shoes.git
    
```
Start


```
    node app.js
```
La web corre en servidor local.<br />
Open [http://localhost:3030](http://localhost:3030) para verla en el navegador.
